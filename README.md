# رواية للزهور — موقع تعريفي

موقع تسويقي عربي (RTL) لمحل **رواية للزهور** في الطائف · حي الجال.

## النشاط والمدينة
- **النشاط:** محل ورد وباقات وصناديق هدايا وتنسيق مناسبات وتوصيل.
- **المدينة:** الطائف (الجال) — منطقة مكة المكرمة.
- **التقييم:** 4.8 من 5 بناءً على 310 تقييمات على خرائط قوقل (مذكور كما هو، دون تعديل).

## لماذا اخترنا هذا المحل
اسم «رواية» يحمل معنى الحكاية، وهي زاوية تسويقية مميزة لمحل ورد: كل باقة تُروى كفصل من مناسبة العميل. هذه القصة تسمح بهوية بصرية ناعمة ومميزة عن باقي محلات الورد.

## فكرة التصميم
أسلوب **«حكاية ناعمة» (Soft Storytelling)**: لوحة ألوان من اللافندر الناعم + الميرمية (السيج) + الكريمي الدافئ، مع نص بلون البرقوق (Plum). خطوط: **El Messiri** للعناوين و**Tajawal** للنص. حركة لطيفة مستوحاة من الزهور (بتلات منجرفة، كين-بيرنز، تفتّح الكروت عند التمرير).

## التقنية
- موقع **ثابت Vanilla** بدون أي خطوة بناء: HTML + CSS + JavaScript فقط.
- لا يوجد `node_modules` ولا `package.json`.
- خطوط Google عبر preconnect + `display=swap`.
- أيقونات SVG inline (بدون إيموجي كأيقونات).

## حالة الـ Backend
**غير متصل بخادم.** نموذج الطلب تجريبي:
- يتحقّق من الحقول، ثم يبني رسالة **واتساب** جاهزة (`wa.me/966507140313`).
- يحفظ نسخة من الطلب في **localStorage** للمتصفح فقط (عرض تجريبي).
- لا توجد قاعدة بيانات ولا إرسال تلقائي.

## مصدر الصور
صور حقيقية من ملف المحل على خرائط قوقل، **منتقاة يدويًا**.
- المُستخدمة: `rw-23057` (هيرو/باقة بالون)، `rw-13177` (رفوف العرض)، `rw-24218` (ركن الاستقبال)، `rw-5538` (تنسيق هدايا المناسبات).
- المُستبعدة: `rw-28111` و`rw-31103` (تحملان لافتات لأسماء محلات أخرى — لا تخص رواية).

## نتائج الاختبارات
Playwright (مشروعان: mobile + desktop) — **18/18 ناجحة**:
- RTL + lang=ar + العنوان صحيح.
- العنوان الرئيسي وزر الـ CTA ظاهران.
- تقييم قوقل مذكور.
- كل صورة لها مصدر صالح و`alt`.
- قائمة الجوال ملء الشاشة تفتح وتغلق.
- لا تمرير أفقي عند 390px.
- تحقّق نموذج الطلب من الحقول.
- ظهور رسالة النجاح (Toast) عند طلب صحيح.
- صفحة 404 فيها زر العودة للرئيسية.
