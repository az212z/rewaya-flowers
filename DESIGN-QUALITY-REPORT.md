# DESIGN-QUALITY-REPORT — رواية للزهور

موقع تعريفي عربي RTL لمحل **رواية للزهور** (الطائف · الجال). أسلوب «حكاية ناعمة».

---

## 1) المهارات المُستدعاة وكيف طُبّقت

### `ui-ux-pro-max` (+ `--design-system`)
شغّلت:
```
python3 ~/.claude/skills/ui-ux-pro-max/scripts/search.py "florist flower shop soft pastel storytelling" --design-system
```
**المخرجات المعتمدة:**
- **Pattern:** *Scroll-Triggered Storytelling* → بنينا الصفحة كحكاية: هيرو يفتح القصة → خدمات → فصل «لماذا رواية» (story) → معرض → طلب → ختام. كل قسم يظهر بـ scroll-reveal.
- **Style:** *Organic Biophilic* (طبيعي، أشكال ناعمة، زوايا مدوّرة، ظلال طبيعية) → زوايا 10–24px، ظلال مائلة للبرقوق، أشكال بتلات.
- **Effects:** زوايا 16–24px، منحنيات عضوية، ظلال ناعمة → مطبّقة في `--r-md/--r-lg` ونظام `--sh-1/2/3`.
- **Checklist:** التزمنا بكامل قائمة ما قبل التسليم (لا إيموجي كأيقونات، cursor، hover 150–300ms، تباين 4.5:1، focus، reduced-motion، استجابة 375→1440).

**انحراف مقصود عن مخرجات السكِربت:** البريف طلب لوحة **لافندر + سيج + كريمي + برقوق**، وهي أرقى وأكثر تميّزًا للنشاط من الأخضر/الوردي الافتراضي (الذي يشبه قوالب الويلنس). كذلك اعتمدنا خطوط البريف (El Messiri + Tajawal) بدل Varela/Nunito لأن الموقع عربي. هذا يحقق «نظام تصميم مستقل لكل مشروع» المطلوب في القواعد.

### `design-taste-frontend` (anti-slop)
- **Design Read:** «موقع تعريفي محلي (florist) لجمهور سعودي، بلغة ناعمة سردية، يميل إلى vanilla CSS + خطوط برند + حركة مضبوطة.»
- **Dials:** VARIANCE 7 (هيرو split غير متماثل، معرض bento بإيقاع متنوّع)، MOTION 6 (بتلات، كين-بيرنز، reveal، sheen)، DENSITY 3 (مساحات واسعة).
- **مكافحة الـ tells:** بلا AI-purple slop (اللافندر هوية مقصودة ومضبوطة)، بلا em-dash نهائيًا، بلا 3 كروت متطابقة (الخدمات 4 ببنية واحدة لكن القسم التالي مختلف)، بلا أسماء وهمية، بلا أسعار مخترعة («حسب الطلب»). الهيرو يسع الشاشة (عنوان سطرين، lead < 20 كلمة). قفل الثيم (light واحد للصفحة كلها)، قفل اللون (اللافندر accent ثابت)، قفل الشكل (سلّم زوايا واحد). صور حقيقية لا fake screenshots.
- **تنويع التخطيط:** هيرو split → trust bar 4 → خدمات grid → story (صورة/نص) → معرض bento → فورم split → موقع 3 بطاقات → ختام مركزي. ≥4 عائلات تخطيط.

### `emil-design-eng` (الموشن والتفاصيل)
- **Easing مخصّص:** `cubic-bezier(0.23,1,0.32,1)` للدخول، لا `ease-in`.
- **مدد:** ضغط الأزرار 160ms، reveal 600ms، toast 420ms — كلها ضمن المنطق (UI تحت ~300ms للميكرو).
- **scale(0.97) على `:active`** لكل زر وFAB (ردّ فعل لمسي).
- **لا scale(0):** الكروت/المعرض تبدأ من `scale(.92–.96)+opacity` (لا شيء يظهر من العدم).
- **transform/opacity فقط** في كل الحركات (أداء GPU، بلا CLS).
- **Stagger 40–60ms** للكروت عبر `data-delay` (IntersectionObserver).
- **CTA sheen** عبر transform فقط.

## 2) قرارات UI/UX الأساسية
- **القصة كبنية:** الأقسام تتبع منطق الحكاية (اسم المحل = «رواية»).
- **هيرو split** غير متماثل: نص يمين + صورة بكين-بيرنز يسار، مع بتلات منجرفة وبادج تقييم قوقل.
- **trust bar** بأربع رقائق تشمل التقييم الحقيقي 4.8 (310).
- **نموذج طلب** يبني رسالة واتساب + localStorage + toast، موسوم بوضوح كتجريبي.
- **FABs** عائمة: واتساب + اتصال + خريطة.
- **lightbox** خفيف للمعرض (Enter/مسافة + Esc + إغلاق بالخلفية).

## 3) سبب الألوان والخطوط
- **اللافندر/البرقوق:** أنوثة راقية وهدوء سردي يناسب الورد دون ابتذال.
- **السيج (الميرمية):** يربط بالطبيعة/الأوراق ويوازن دفء الكريمي.
- **الكريمي الدافئ:** خلفية مريحة بصريًا تبرز صور الورد.
- **El Messiri:** خط عربي عرضي أنيق ذو طابع سردي للعناوين.
- **Tajawal:** خط نظيف عالي القابلية للقراءة للنص (≥16px).

## 4) إمكانية الوصول (Accessibility)
تحقّقت فعليًا عبر preview inspect:
- تباين: lead **7.15:1**، H1 **8.97:1**، زر primary **5.89:1** (كلها > 4.5:1).
- HTML سيمانتيك: header/nav/main/section/footer، تدرّج h1→h2→h3 بلا قفز.
- كل صورة لها `alt` عربي + `width/height` + `loading=lazy` (عدا الهيرو) + `decoding=async`.
- كل زر أيقونة له `aria-label`؛ القائمة واللايتبوكس `role=dialog` + `aria-modal` + `aria-hidden` متبدّل.
- `:focus-visible` بحدّ 3px على كل عنصر تفاعلي.
- `aria-live=polite` لرسائل الخطأ والـtoast؛ Esc يغلق القائمة واللايتبوكس.
- `prefers-reduced-motion`: يوقف البتلات والكين-بيرنز ويُظهر كل المحتوى فورًا.

## 5) Layout & Responsive
- Mobile-first؛ تحقّق: **لا تمرير أفقي عند 390px** (scrollWidth == clientWidth).
- **قائمة الجوال = overlay ملء الشاشة** (مُتحقَّق: 390×844، خلفية صلبة `rgb(251,247,242)` غير شفافة، زر X واضح).
- `min-h-dvh`/`100dvh` لا `100vh`؛ breakpoints 560/620/680/760/820/920/980/1000.

## 6) الحركة (Motion)
- بتلات هيرو (9، transform فقط، تُحقن JS وتُخفى عند reduced-motion).
- كين-بيرنز بطيء للهيرو (22s alternate).
- تفتّح الكروت/المعرض على التمرير: `scale .92→1 + fade + stagger 40–60ms` عبر IntersectionObserver + fallback أمان 2.5s.
- hover image zoom 1.03–1.05 + رفع ظل.
- CTA sheen + ضغط scale(0.97).
- حد أقصى عنصر/عنصرين متحركان لكل مشهد.

## 7) النصوص
- عربية سعودية طبيعية، **محايدة جندريًا** (اطلب/تواصل/شاهد — بلا اطلبي/لكِ).
- لا أسعار مخترعة («حسب الطلب»). التقييم الحقيقي من قوقل فقط (4.8 / 310). النموذج موسوم تجريبيًا.
- بلا em-dash، بلا كلام روبوتي، بلا وعود مبالغ.

## 8) اختبار الذوق (Taste / Impeccable)
- فاخر؟ نعم — لافندر/سيج/كريمي + خطوط راقية + ظلال ناعمة.
- سعودي مناسب للنشاط؟ نعم — لهجة وصور وهوية محل ورد طائفي.
- يقنع خلال 3 ثوانٍ؟ نعم — عنوان واضح + صورة + تقييم + CTA فوق الطية.
- لا يشبه قالبًا مجانيًا؟ نعم — نظام تصميم مخصّص ومتميّز عن المواقع الشقيقة.
